# 🔐 Python Port Scanner

A Python-based port scanner designed to identify open ports on a target system. This tool is useful for basic network reconnaissance in penetration testing.

---

## 🚀 Features
- Scans commonly used ports
- Identifies open ports on a target
- Handles errors like unreachable hosts

---

## 🛠️ Tech Stack
- Python
- Socket Programming

---

## ▶️ Usage

```bash
python port_scanner.py
